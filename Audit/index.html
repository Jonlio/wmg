Audit

Web More Green utilise des outils qui permettent d'évaluer:

  • L'empreinte technique de chaque page (poids, complexité, taille du DOM...)
  • Le score environnementale (note de 0 à 100)
  • L'empreinte environnementale de chaque connexion à votre site web (gaz à effet de serre et consomation d'eau)

Le but de cet audit est de prendre conscience de l'impact environnementale de votre site web. Il permettra également d'identifier les causes de cet impact.

Web More Green vous proposera des solutions concrètes pour réduire ces impacts.
