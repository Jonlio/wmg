Refonte

Le principe de base de Web More Green? Faire au moins aussi bien, mais avec moins!

La refonte (ou refectorisation) de votre site web sera basée sur 3 axes principaux:

  • Réduire le nombre de requêtes (allers retours navigateur/serveur)
  • Diminuer le poids de votre site web (code inutilisé, images non compressées...)
  • Limiter le traitement d'information

Les bonnes pratiques d'eco-conception web seront mises en place pour lutter contre "l'obesiciel", qui touche pratiquement tous les sites web.
